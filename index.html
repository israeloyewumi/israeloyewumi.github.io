ABOUT ME

Hi! I'm Israel Oyewumi, a passionate app developer based in Jos, Plateau State, Nigeria. I specialize in creating innovative applications that solve real-world problems and make a tangible difference in people's lives.

My journey into software development began with a fascination for how technology can transform education. I believe that great software should be intuitive, efficient, and purpose-driven. I build robust applications using modern technologies like React, React Native, Electron, and TypeScript to deliver seamless experiences across desktop and mobile platforms.

I'm particularly interested in educational technology and creating tools that empower teachers and students. My work demonstrates a commitment to solving practical challenges in the education sector through thoughtful software design and full-stack development.

When I'm not coding, I'm exploring new technologies, learning best practices in software architecture, and thinking about how to make my next application even better than the last.

MY PROJECTS

EduApp

Offline Educational Platform for Windows

A comprehensive desktop application that revolutionizes assessment management in schools. Teachers can create, distribute, and automatically grade assessments while students receive instant feedback.

KEY FEATURES

  • Dual dashboard system for teachers and students
  • Create and export comprehensive assessments
  • Automatic grading with instant feedback
  • Performance tracking and analytics
  • Support for multiple question types
  • Works completely offline

IMPACT

  • Eliminates paper waste and printing costs
  • Saves teachers hours on manual grading
  • Provides immediate student feedback
  • Enables comprehensive performance tracking
